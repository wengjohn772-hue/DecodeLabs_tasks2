##  Project Overview
Overview

This project demonstrates the implementation of a basic machine learning classification model using the Iris dataset. The objective is to predict the species of an iris flower based on its physical measurements. The project introduces fundamental concepts of supervised learning, data preprocessing, model training, prediction, and evaluation.

----

## Project Task
Load and explore a dataset.
Understand dataset features and target variables.
Split data into training and testing sets.
Train a classification model.
Evaluate model performance.

----

## Data Sourcing
The dataset was loaded using Scikit-Learn's built-in dataset library.

Dataset

The Iris dataset is one of the most widely used datasets in machine learning. It contains 150 flower samples belonging to three species:

Iris Setosa
Iris Versicolor
Iris Virginica

### Features
Sepal Length (cm)
Sepal Width (cm)
Petal Length (cm)
Petal Width (cm)
